# react-demo

A profile demo built with React + Flux. [Do It Live!](https://ambitioninc.github.com/react-demo)

## Install

To install using npm:
```shell
npm install ambitioninc/react-demo
```

## Building

To rebuild the static files:
```shell
npm install
npm run build
```
